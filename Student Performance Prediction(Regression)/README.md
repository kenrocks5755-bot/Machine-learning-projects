# Student Performance Prediction

This project predicts student exam marks using multiple regression models.

## 📌 Problem Statement
Predict student performance based on study habits and lifestyle factors.

## 📊 Features
- Hours Studied
- Sleep Hours
- Class Attendance
- Practice Tests
- Phone Usage

## 🛠️ Models Used
- Linear Regression
- Ridge Regression
- Lasso Regression

## 📈 Evaluation
Mean Squared Error (MSE) is used to compare models.
